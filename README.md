# FUTURE_DS_01 — Superstore Sales Performance Dashboard

## Task Overview
Business Sales Performance Analytics using the Superstore Sales dataset.
Built as part of the Future Interns Data Science & Analytics Internship (2026).

## Tool Used
- Tableau

## Dataset
- Superstore Sales Dataset (Kaggle)

## Files in this Repository
- `superstore.twb` — Tableau workbook file
- `Superstore_Report.docx` — Full analytics report with insights and recommendations
- `Superstore_dashboard.png` — Dashboard screenshot

## Key Insights
- Technology is the most profitable category
- Sales peak significantly in November each year
- The Consumer segment drives the highest purchase volumes
- Furniture category needs urgent pricing review
