📊 FUTURE INTERNS – DATA SCIENCE & ANALYTICS TASK 01

🧠 Task Overview
This project was completed as part of the Future Interns Data Science & Analytics track (DS).

The objective was to analyze business sales data and build a dashboard that provides meaningful insights for business decision-making.

📊 Dashboard Preview
Dashboard

🔍 Key Insights
Technology category generates the highest profit  
Furniture has high sales but low profit margins  
Sales show a consistent upward trend over time  
High discounts reduce profitability significantly  
Standard shipping dominates sales  
West region performs best in both sales and profit  

💡 Recommendations
Focus on high-performing categories  
Improve pricing strategy for low-margin products  
Reduce excessive discounting  
Invest in top-performing regions
